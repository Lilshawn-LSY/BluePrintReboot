# BluePrint Reboot

Fresh Streamlit project scaffold.

## Layout

- `app.py` - Streamlit entrypoint.
- `ui_streamlit/` - Streamlit UI code.
- `core/` - Core application logic.
- `storage/` - Persistence and file storage helpers.
- `ingest/` - Data and document ingestion helpers.
- `tests/` - Test suite.
- `data/`, `papers/`, `notes/`, `exports/` - Local project workspaces.

## Run

```powershell
streamlit run app.py
```
