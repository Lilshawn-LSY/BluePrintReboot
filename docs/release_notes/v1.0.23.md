- Added a Streamlit-free Reading Note state contract with paper-scoped clean/dirty derivation and idempotent pending events.
- Added visible Saved, Unsaved changes, and Header refresh pending status near the editor.

- Ensured saved header refresh baselines do not incorrectly mark a later edited draft clean.
